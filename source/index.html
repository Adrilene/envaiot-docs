EnvAIoT

The Environment for modelling and simulating Adaptive Internet of Things applications tool assists IoT system designers, engineers, architects, and developers in the design phase of development by focusing on the system's self-adaptation feature and providing an approach to model and validate if their adaptation strategies are effective. EnvAIoT considers that an IoT system contains several scenarios, which are critical conditions in which the system may be. It operates in two stages: Modeling and Simulating. In the former, the user models many features of the system, such as the component devices and scenarios, using a DSL, whereas, in the latter, the tool simulates the devices, interactions, and adaptation based on the model that was built in the first phase.

This documentation details how each component works and how to use the tool. You can download the postman collection below Download Collection
